## Descrição do Projeto

Este projeto de automação serve como base de estudo para automação de API's utilizando plawright + nodejs.

## Pré-requisitos

Certifique-se de ter os seguintes pré-requisitos instalados em seu ambiente de desenvolvimento:

- Node (versão recomendada 18.9.0)
- NPM
- IDE (Vscode)

## Como Executar?

- Localmente via IDE

## Instalação
1. Clone o repositório: `git clone https://github.com/arturoribeira/playwright-nodejs-api`
2. Navegue até o diretório: `cd playwright-nodejs`
3. Instale as dependências: `npm install`

Para execução de todos os cenários utilizar o comando:
npx playwright test --grep=@allTests

Para visualização do report allure:
allure serve allure-results
